# logic gates project using the app Blynk

- Flexible logic gate configuration. Add as many gates as virtual pins.
- Virtual pins on the app's widgets:

  input A = vpin 1

  input B = vpin 2

  gate menu = vpin 3

  3x3 table = vpin 4

  output = vpin 6
